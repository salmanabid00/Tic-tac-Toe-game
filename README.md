# Tic-Tac-Toe Game

A classic, fully functional Tic-Tac-Toe game built with HTML, CSS, and JavaScript. 

## 🎮 Play the Game
This is a local web application. Simply open the `index.html` file in any modern web browser to start playing.

## ✨ Features
*   **Two-Player Mode:** Play against a friend locally (Player O vs Player X).
*   **Win Detection:** Automatically detects when a player gets 3 in a row (horizontally, vertically, or diagonally) and announces the winner.
*   **Interactive UI:** Clicking a box disables it to prevent overwriting moves.
*   **Reset & New Game:** Includes a "Reset Game" button to clear the board mid-game, and a "New Game" button after someone wins.
*   **Responsive Design:** Fully optimized layout that scales perfectly across desktop, tablet, and mobile devices.

## 🛠️ Technologies Used
*   **HTML5:** Structured the game board and message containers.
*   **CSS3:** Styled the layout using Flexbox, `vmin` units for proportional scaling, and custom media queries for responsiveness.
*   **JavaScript (Vanilla):** Implemented the game logic, turn switching, win-pattern validation, and DOM manipulation.

## 🚀 How to Use
1.  Clone or download the repository.
2.  Open the folder containing the project files.
3.  Double-click `index.html` to open it in your browser.
4.  Player "O" starts first. Click any empty box to make your move.
5.  Take turns until someone wins or the board is full.
6.  Click **Reset Game** anytime to restart, or **New Game** when the match concludes.
